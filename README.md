# How to start project?
* Backend & database connection
  - Open backend folder by: 
      ```console
      cd backend
      ```
  - Install all of the packages by:
      ```console
      npm i
      ```
  - Strt server by:
    ```console
    npm start`
    ```
* Frontend 
  - Open backend folder by:
     ```cmd
     cd frontend
     ```
  - Start fontend by: `ng serve`
