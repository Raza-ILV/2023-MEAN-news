# How to start project?
* Backend & database connection
  - Open backend folder by: 
      ```cmd
      cd backend
      ```
  - Install all of the packages by:
      ```cmd
      npm i
      ```
  - Start server by:
    ```cmd
    npm start
    ```
* Frontend 
  - Open backend folder by:
     ```cmd
     cd frontend
     ```
  - Start fontend by:
    ```cmd
    ng serve
    ```
* Open in browser
    `http://localhost:4200`
